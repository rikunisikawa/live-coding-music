# プライバシーポリシー

最終更新日: 2026-02-21

`live-coding-music-dev`（以下「本アプリ」）における、ユーザー情報の取扱いを以下のとおり定めます。

## 1. 取得する情報
本アプリは Fitbit Web API を通じて、ユーザーが許可した範囲で以下の情報を取得します。
- 活動データ（steps）
- 心拍データ（heartrate）
- 睡眠データ（sleep）
- OAuth 認証情報（access token、refresh token）

## 2. 利用目的
取得した情報は、以下の目的でのみ利用します。
- 本アプリ機能の提供（データ取得、表示、分析）
- 不具合調査および運用改善

## 3. 第三者提供
法令に基づく場合を除き、取得した個人データを本人の同意なく第三者へ提供しません。

## 4. 保存場所と安全管理
- 取得データおよび token は、開発者管理環境（ローカルまたは指定ストレージ）に保存されます。
- 不正アクセス、漏えい、改ざん、滅失の防止に合理的な安全管理措置を講じます。

## 5. 保存期間
取得データは、利用目的達成に必要な期間のみ保持し、不要になったデータは適切に削除します。

## 6. ユーザーの権利
ユーザーは、自己に関するデータの開示、訂正、削除を求めることができます。希望する場合は下記連絡先まで連絡してください。

## 7. 外部サービス
本アプリは Fitbit プラットフォームおよび API を利用します。Fitbit 側でのデータ取扱いは Fitbit の規約およびポリシーに従います。

## 8. 改定
本ポリシーは必要に応じて改定することがあります。改定後は公開時点から適用されます。
