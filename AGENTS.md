# AI エージェント規約

すべてのAIエージェントは、コード生成前に必ず `ai/instructions.md` を読むこと。

本リポジトリで作成するドキュメントはすべて日本語とする。

Skills のリポジトリ内正本は `.agents/skills/` とする。

## ブランチ運用ルール
- 1 Spec につき 1 ブランチを必須とする（`1spec = 1branch`）。
- 実装着手前に、対象 Spec ID を含む専用ブランチを作成する。
- 1ブランチで複数Specを同時に進めない。
- マージは Step 3（テスト）と Step 4（レビュー）完了後のみ許可する。
